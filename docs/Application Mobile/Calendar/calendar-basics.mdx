# Les bases du calendrier 
## _Utiliser les filtres, les vues et déplacer des événements._

Le calendrier est un module essentiel de MyCoach Pro. 
Il vous permet de créer des événements, de les planifier, les consulter ou encore de les supprimer.  
Pour que votre utilisation au quotidien soit plus simple et optimisée pour plus d'efficience, 
vous avez la possibilité de filtrer les événements de différentes manières, mais aussi d'utiliser différentes vues selon votre besoin.

1\. Vous pouvez **déplacer un événement** via un **drag and drop**.

*Cela ne fonctionnera pas dans les cas suivants:*

- ***Les anniversaires et matchs officiels.***
- ***Les événements compotant une réservation d'infrastructure(s).***
- ***Vous n'êtes pas propriétaire de l'événement.***

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-14/04bea8dc-28a0-49c2-b878-29c1e28f4c8a/ascreenshot.jpeg?tl_px=47,111&br_px=820,544&force_format=png&width=774&wat_scale=69&wat=1&wat_opacity=1&wat_gravity=northwest&wat_url=https://colony-recorder.s3.amazonaws.com/images/watermarks/F43F5E_standard.png&wat_pad=362,191)


2\. *Si ce dernier a été planifié, il ne sera plus lié à la récurrence d'événements.*

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-14/10455bcd-6d71-4d9a-bcf7-98386ccab22d/user_cropped_screenshot.jpeg?tl_px=0,11&br_px=663,444&force_format=png&width=774&wat_scale=69&wat=1&wat_opacity=1&wat_gravity=northwest&wat_url=https://colony-recorder.s3.amazonaws.com/images/watermarks/F43F5E_standard.png&wat_pad=623,373)


3\. Cliquer sur un événement vous permet d'afficher une **liste d'options disponibles**.\
 (*Elle peut différer selon le type d'événement.*) 

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-14/ced01e04-ae90-4e84-9d87-42ef27bdd015/screenshot.gif)


4\. Différentes vues sont disponibles pour votre planning:

- **Mensuel**
- **Hebdomadaire**
- **Jour**
- **Week-End**
- **Planning**
- **Liste**

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-14/6ff39b8e-7d65-4543-9651-1199d71418bd/screenshot.gif)


5\. Le calendrier **JOUEURS** vous permet de **consulter les événements auxquels un joueur est convoqué**, il est **disponible dans toutes les vues**, et vous **indique son statut de présence**.

Cliquez sur l'onglet JOUEURS, recherchez le joueur et cliquez sur son nom pour l'afficher.

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-14/fda72761-c728-492a-876f-7f33d139598f/user_cropped_screenshot.jpeg?tl_px=157,119&br_px=1017,600&force_format=png&width=860)


6\. Le calendrier **STAFFS** vous permet de **consulter les événements créés par un staff**, il est **disponible dans toutes les vues**.

Cliquez sur l'onglet STAFFS, recherchez le staff et cliquez sur son nom pour l'afficher.

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-14/7abac63d-f068-4415-8f40-979070ab5a8d/screenshot.png?tl_px=304,50&br_px=1164,531&force_format=png&width=860)


7\. Vous pouvez **utilisez les filtres pour optimiser la vue de votre calendrier**.

Cliquez sur le bouton dans le coin supérieur droit pour accéder aux filtres.

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-14/460f583d-ff3e-412c-ad3c-9145122d5cea/ascreenshot.jpeg?tl_px=1146,0&br_px=1920,432&force_format=png&width=774&wat_scale=69&wat=1&wat_opacity=1&wat_gravity=northwest&wat_url=https://colony-recorder.s3.amazonaws.com/images/watermarks/F43F5E_standard.png&wat_pad=733,-13)


8\. Vous pourrez **décocher les case pour masquer**:

- **Les types d'événements**
- **Les équipes**
- **Les types de séances**

Vous pouvez également choisir d'**afficher une couleur par type de séance, d'événement, ou une couleur par équipe**.
