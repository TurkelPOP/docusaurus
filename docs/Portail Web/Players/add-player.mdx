**1\.** Cliquez sur le module "**Joueurs**".

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-10-09/5ce98701-15b6-4b99-9e5b-eefd8eb48470/ascreenshot.jpeg?tl_px=0,0&br_px=859,480&force_format=png&width=860&wat_scale=76&wat=1&wat_opacity=1&wat_gravity=northwest&wat_url=https://colony-recorder.s3.amazonaws.com/images/watermarks/F43F5E_standard.png&wat_pad=19,157)

**2\.** Puis cliquez sur le bouton "**Ajouter un joueur**".

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-10-09/0a906ff7-0ea1-4418-950f-4acc4be13679/ascreenshot.jpeg?tl_px=1700,0&br_px=2560,480&force_format=png&width=860&wat_scale=76&wat=1&wat_opacity=1&wat_gravity=northwest&wat_url=https://colony-recorder.s3.amazonaws.com/images/watermarks/F43F5E_standard.png&wat_pad=775,54)

**3\.** Remplissez les différents champs disponible, attention certains sont **obligatoires**, comme **l'identifiant**, le **mot de passe**, et le **poste du joueur**.  
**ATTENTION: un identifiant MyCoach Pro est unique, il ne peut être utilisé pour plusieurs comptes différents.**

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-10-09/9a9d04cb-ccad-4ec2-8a46-7decd1ccdc27/ascreenshot.jpeg?tl_px=0,282&br_px=859,763&force_format=png&width=860&wat_scale=76&wat=1&wat_opacity=1&wat_gravity=northwest&wat_url=https://colony-recorder.s3.amazonaws.com/images/watermarks/F43F5E_standard.png&wat_pad=398,212)

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-10-09/bab38ca5-925f-40c6-9c52-6916dd1cb34e/ascreenshot.jpeg?tl_px=0,349&br_px=859,830&force_format=png&width=860&wat_scale=76&wat=1&wat_opacity=1&wat_gravity=northwest&wat_url=https://colony-recorder.s3.amazonaws.com/images/watermarks/F43F5E_standard.png&wat_pad=376,212)

**5\.** **Attention**, les mots de passe sont standardisés selon les critères discriminants suivants:

- **8 caractère**
- **1 minuscule**
- **1 majuscule** -**1 chiffre**
- ** caractère spécial (@#%$&%\*!...)**

**9\.** **IMPORTANT:** Sélectionnez les **modules auxquels vous souhaitez donner accès à vos joueurs**. Exemple ici, **le module séance joueur** leur permet de **créer des séance depuis l'application MyCoach Pro - AMS.**

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-10-09/7ed8b227-22d2-4218-aa39-bb611d92be76/user_cropped_screenshot.jpeg?tl_px=0,22&br_px=293,318&force_format=png&width=455)

**10\.** Une fois la fiche remplie à votre convenance, cliquez sur "**TERMINÉ**".\
Vous retrouverez votre joueur dans la liste de vos effectifs.
