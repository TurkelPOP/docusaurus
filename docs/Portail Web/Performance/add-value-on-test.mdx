# Ajouter une valeur sur un test athlétique

#### [Made by Lucas Poirel with Scribe](https://scribehow.com/shared/Ajouter_une_valeur_sur_un_test_athltique___vaAbw4WTdGhMjfAxfqwFA)

1\. Cliquez sur le module **Performance &gt; Suivi Athlétique**

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-10-31/c12fb78f-9984-44c0-ab6c-45f258c0451b/ascreenshot.jpeg?tl_px=0,631&br_px=1376,1400&force_format=png&width=1120.0&wat=1&wat_opacity=1&wat_gravity=northwest&wat_url=https://colony-recorder.s3.amazonaws.com/images/watermarks/F43F5E_standard.png&wat_pad=331,277)

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-10-31/68a3b3b3-e687-4901-875b-77335c96d690/ascreenshot.jpeg?tl_px=0,796&br_px=1376,1566&force_format=png&width=1120.0&wat=1&wat_opacity=1&wat_gravity=northwest&wat_url=https://colony-recorder.s3.amazonaws.com/images/watermarks/F43F5E_standard.png&wat_pad=165,288)

3\. Vous pouvez entrer une valeur sur un test pour un joueur en utilisant le champs de texte pour la date du jour.

Pour une date antérieure, cliquez sur le bouton **\+** .

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-10-31/27d45e32-42dd-4113-8ef6-efd484e70acc/ascreenshot.jpeg?tl_px=794,530&br_px=2171,1299&force_format=png&width=1120.0&wat=1&wat_opacity=1&wat_gravity=northwest&wat_url=https://colony-recorder.s3.amazonaws.com/images/watermarks/F43F5E_standard.png&wat_pad=524,277)

4\. Choisissez **une date** et **entrez une valeur**, puis cliquez sur **AJOUTER**

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-11-29/24b90e26-2def-4546-9a31-696057810bea/screenshot.png?tl_px=0,24&br_px=557,505&force_format=png&width=860)

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-11-29/b578b926-d7cd-44e7-9c18-e56b5e11dda1/screenshot.png?tl_px=0,39&br_px=574,520&force_format=png&width=860)

6\. Vous pouvez également ajouter pour un joueur tous les tests effectués en une seule fois en cliquant sur le bouton **AJOUTER DES VALEURS**.

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-11-29/3f5a2194-d010-4124-b78a-0468ad1a6f95/user_cropped_screenshot.jpeg?tl_px=0,0&br_px=1281,461&force_format=png&width=1120.0)

7\. Choisissez une date.

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-11-29/e755b894-e0e1-485e-9e54-d3164f8d1fcd/user_cropped_screenshot.jpeg?tl_px=0,64&br_px=909,834&force_format=png&width=1120.0)

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-10-31/ae016304-7ce7-457d-8e0f-6858db105fbe/ascreenshot.jpeg?tl_px=623,626&br_px=2000,1395&force_format=png&width=1120.0&wat=1&wat_opacity=1&wat_gravity=northwest&wat_url=https://colony-recorder.s3.amazonaws.com/images/watermarks/F43F5E_standard.png&wat_pad=523,277)

9\. Entrez la valeur des tests, puis cliquez sur **ENREGISTRER**.

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-10-31/6045d3b8-07fa-4995-aee0-e61a8069cc95/ascreenshot.jpeg?tl_px=1050,395&br_px=2427,1164&force_format=png&width=1120.0&wat=1&wat_opacity=1&wat_gravity=northwest&wat_url=https://colony-recorder.s3.amazonaws.com/images/watermarks/F43F5E_standard.png&wat_pad=524,277)
